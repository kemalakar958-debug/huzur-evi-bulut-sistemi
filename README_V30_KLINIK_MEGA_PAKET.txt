v30 Klinik Mega Paket

Bu tek büyük paket parça parça kurulum yorgunluğunu azaltmak için hazırlandı.

İçerir:
- Vital Takip
- Olay / Düşme Kaydı
- Hastane Sevk
- MAR İlaç Uygulama
- Görev Takip
- Nöbet Teslim
- Bakım Planı
- Günlük Bakım
- Beslenme / Sıvı Takibi
- Bası / Pansuman Takibi
- Randevu / Doktor Takibi
- Sol menü ve mobil menü güncellemesi

İçermez:
- Finans / muhasebe
- WhatsApp
- Yapay zeka
- Barkod

Kurulum:
1. Zip'i aç.
2. app, components ve supabase klasörlerini GitHub projenin ana dizinine yükle.
3. Aynı dosyalar varsa üzerine yaz.
4. GitHub'da Commit changes yap.
5. Supabase > SQL Editor > New Query aç.
6. supabase/schema_v30_clinical_mega.sql içindeki SQL'i çalıştır.
7. Vercel deploy Ready olunca siteyi yenile.

Test sırası:
1. Vital Takip kaydı gir.
2. Olay / Düşme kaydı gir.
3. Hastane Sevk kaydı gir.
4. MAR kaydı gir.
5. Görev oluştur.
6. Nöbet teslim kaydı oluştur.
7. Bakım planı ve günlük bakım kaydı gir.
